# -Fraud-Detection-Application-in-internship

Objective:
Detect fake internship applications using Machine Learning.

Algorithm:
- Isolation Forest

Features Used:
- Submission Time
- Duplicate Count
- Age
- CGPA

Output:
- Detect suspicious applications
- Save results in Fraud_Detection_Results.csv
- Visualization of fraud vs normal applications

Libraries:
- pandas
- numpy
- scikit-learn
- matplotlib
